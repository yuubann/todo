Simple Todo App

1.アプリケーションの説明
　Javaの基礎学習として作成した、コマンドラインで動作するタスク管理アプリケーションです。オブジェクト指向に基づいた設計を行っており、実用的なデータの永続化の機能も備えています。

2.概要
　オブジェクト指向の基本である、カプセル化やコレクション(List)を学び、
　メンテナンス性の高いコードを書くことを目的として制作しました。

3.特徴
　 リファタリングの実施 : 
　  -mainメソッドをスッキリさせるため、処理ロジックをTodoProcessクラスに分離しました。
　  -マジックナンバーを避け、ユーザー入力の番号とリストのインデックス調整を行いました。
　 堅牢な設定 : 
 　 -入力エラー(数字以外の入力など)に対する例外処理を実装しました。
　 カプセル化 : 
 　 -フィールドをprivateに保ち、メソッド経由で状態を安全に変更できるようにしました。
 　 -データ保持(Task)、ロジック(Service)、実行(Todo)で役割を分担しました。
 　デコレーターパターンの活用：
　　-FileOutputStreamをOutputStreamWriterやBufferedWriterで包むことで柔軟なファイル操作を実現しました。

4.使い方
　Windowsのコマンドプロンプトで日本語を正しく表示するために、以下の手順で実行してください。
　○コンパイル
 　 javac -encoding UTF-8 *.java
　○文字コードの設定
　　chcp 65001
　○実行
 　 java -Dfile.encoding=UTF-8 Todo
